# ميزات التطبيق - دليل المستخدم

## الميزات المُفعّلة

### 1. الإعدادات المتقدمة ✅
- **المعلومات الشخصية**: اسم المستخدم، البريد الإلكتروني، رقم الهاتف، الصورة الرمزية
- **معلومات المشروع**: اسم المشروع، اسم الشركة، الموقع، جهة الاتصال
- **إعدادات العمل والسلامة**: ساعات العمل اليومية، عدد العمال
- **إعدادات المظهر**: السمة (فاتح/داكن/تلقائي)، اللغة (عربي/إنجليزي)، حجم الخط، تنسيق التاريخ

### 2. إدارة البيانات ✅
- **النسخ الاحتياطي التلقائي**: يمكن تفعيله من الإعدادات
- **النسخ الاحتياطي اليدوي**: زر سحابة سحابة في الشريط العلوي
- **تصدير البيانات**: تصدير جميع البيانات كملف JSON
- **استيراد البيانات**: استيراد ملف JSON لاستعادة البيانات
- **حذف البيانات**: حذف جميع البيانات مع تأكيد مزدوج

### 3. العمل أوفلاين ✅
- **دعم PWA**: التطبيق يمكن تثبيته كتطبيق
- **حفظ البيانات محلياً**: البيانات محفوظة حتى بدون إنترنت
- **عرض حالة الاتصال**: عرض حالة الاتصال في الشريط العلوي
- **تنبيه عند انقطع الإنترنت**: إشعار عند فقدان الاتصال

### 4. الأرشيف الشامل ✅
- **أرشيف التقارير**: عرض جميع تقارير التفتيش
- **أرشيف المخالفات**: عرض جميع المخالفات
- **البحث**: البحث في جميع السجلات
- **التصفية**: تصفية بحسب التاريخ، الحالة، الشدة، إلخ

### 5. المشاركة ✅
- **مشاركة عبر واتساب**: مشاركة التقارير والمخالفات مع نص منسق
- **نسخ النص**: نسخ نص التقرير للحافظة
- **التحميل**: تحميل التقارير كـ PDF (قيد التطوير)

### 6. إدارة المخالفات ✅
- **تسجيل المخالفات**: إضافة مخالفات جديدة مع الصور
- **تحديث الحالة**: إغلاق المخالفات المفتوحة
- **الحذف**: حذف المخالفات مع تأكيد
- **البحث والتصفية**: البحث والتصفية حسب الشدة، الحالة، الفئة

### 7. إدارة التصاريح ✅
- **إصدار التصاريح**: إنشاء تصاريح عمل
- **الاعتماد**: اعتماد التصاريح المعلقة
- **إلغاء التصاريح**: إلغاء التصاريح المنتهية
- **عرض التصاريح**: قائمة بجميع التصاريح مع حالتها

### 8. إحصائيات متقدمة ✅
- **لوحة التحكم**: إحصائيات شاملة
- **رسوم بيانية**: رسم بياني للبيانات
- **مقاييس الأمان**: ساعات الأمان المتراكمة، معدل الامتثال
- **التنبيهات**: تنبيهات المخالفات والتصاريح

## كيفية الاستخدام

### التصدير والاستيراد
1. **النسخ الاحتياطي**:
   - من الإعدادات، فعّل النسخ الاحتياطي التلقائي
   - أو اضغط على زر السحابة السحابة في الشريط العلوي

2. **تصدير البيانات**:
   - من الإعدادات → البيانات → تصدير البيانات
   - سيتم تحميل ملف JSON يحتوي جميع البيانات

3. **استيراد البيانات**:
   - من الإعدادات → البيانات → استيراد البيانات
   - اختر ملف النسخ الاحتياطي
   - سيتم استبدال جميع البيانات الحالية

### العمل أوفلاين
1. التطبيق يعمل بشكل كامل بدون إنترنت
2. البيانات محفوظة محلياً
3. عند فقدان الاتصال، يظهر إشعار
4. حالة الاتصال تظهر في الشريط العلوي (متصل/غير متصل)

### أرشيف التقارير والمخالفات
1. من القائمة السفلية → المزيد → أرشيف التقارير
2. تبويب "التقارير": عرض جميع تقارير التفتيش
3. تبويب "المخالفات": عرض جميع المخالفات
4. استخدم البحث للعثور على سجلات معينة

## الميزات القادمة
- إشعارات فورية عند حدوث أحداث
- نظام إدارة الصور
- دعم تعدد اللغات
- تصميم متجاوب
- حفظ البيانات تلقائياً
- تتبع الإحصائيات اليومية
- تقارير شاملة ومفصلة

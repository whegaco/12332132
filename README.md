# 🛡️ HSE SafetyPro Enterprise

نظام متكامل لإدارة الصحة والسلامة المهنية - Next.js 16 + TypeScript + PostgreSQL

## ✨ الميزات

- 📊 لوحة تحكم شاملة
- ⚠️ إدارة المخالفات والتصاريح
- 🔍 فحوص السلامة الشاملة
- 👥 إدارة اجتماعات الأمان (TBT)
- 🚨 تسجيل الحوادث الوشيكة
- 📈 تقارير وإحصائيات متقدمة
- 💾 تصدير واستيراد البيانات
- 🔄 النسخ الاحتياطي التلقائي
- 📱 دعم العمل بدون إنترنت (PWA)
- 🌙 دعم الوضع الليلي
- 🌍 دعم كامل للعربية

## 🚀 التشغيل المحلي

```bash
# تثبيت الاعتمادات
npm install

# تشغيل التطبيق
npm run dev

# أو باستخدام bun
bun install
bun run dev
```

## 📦 النشر

راجع [DEPLOY_GUIDE_AR.md](./DEPLOY_GUIDE_AR.md) للنشر على Vercel.

## 🛠️ التقنيات

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5
- **Database**: PostgreSQL (Neon)
- **ORM**: Prisma
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **State**: Zustand + TanStack Query
